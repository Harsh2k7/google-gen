# 🚀 Multi-Agent Productivity Assistant

## 📌 Overview

Multi-Agent Productivity Assistant ek AI-based system hai jo multiple agents use karke tasks, scheduling aur workflows automate karta hai.
Yeh system ek smart assistant team ki tarah kaam karta hai jo user ke request ko samajh kar automatically complete karta hai.

---

## 🎯 Problem Statement

Aaj ke time me users ko:

* Tasks manage karne me problem hoti hai
* Scheduling manually karni padti hai
* Multiple apps use karne padte hai

---

## 💡 Solution

Humne ek **multi-agent AI system** banaya hai jisme:

* Main Agent (Orchestrator)
* Task Agent
* Calendar Agent

Ye sab milkar:

* Tasks manage karte hai
* Meetings schedule karte hai
* Data store karte hai
* Complete workflows execute karte hai

---

## ⚙️ Features

* Multi-Agent Architecture
* Task Management System
* Smart Scheduling
* API-Based System (FastAPI)
* Database Integration (SQLite / AlloyDB AI)
* MCP Tool Integration
* Scalable & Modular Design

---

## 🏗️ Architecture

System me:

* Main Agent sab control karta hai
* Sub-agents kaam execute karte hai
* Database me data store hota hai
* MCP tools connect hote hai

---

## 🔄 Workflow

1. User request deta hai
2. Main Agent analyze karta hai
3. Task ko steps me divide karta hai
4. Agents kaam execute karte hai
5. Data database me store hota hai
6. Final output user ko milta hai

---

## 🛠️ Tech Stack
